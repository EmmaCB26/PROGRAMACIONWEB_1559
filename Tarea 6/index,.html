Conceptos principales de HTML

HTML

Lenguaje de marcado de Hipertexto, que se utiliza para hacer o estructurar una página web.

Elemetno HTML

Es la unidad básica de estructura y contenido de una página web, esta compuesta de:

  1. Etiqueta de apertura
  2. Pares atributo-valor
  3. Conteido
  4. Etiqueta de cierre

Etiqueta <head>

Es una etiqueta que contiene metadatos, esla forma en la que una página web se comunica con el navegador, funcione para el SEO, se puede enlazar achivos CSS o Scripts.

Etiqueta <body>

Contiene todo el contenido HTML.

Contenedores Semanticos

<Header> Etiqueta utilizada para el contenido introductorio de una página web, puede contener el logo, el menú de navegación principal, el nombre de la empresa, etc...
<nav> Etiqueta utilizada para el menu de navegacion principal, suele contener enlaces a otras paginas relacionadas o navegar por la pagina principal, no debe de haber mas en la pagina.
<main> Etiqueta unica en una pagina web cuyo proposito es resaltar nuestro objetivo o mensaje principal del sitio web. Dentro se colocara el contenido principal de nuestra pagina
<section> Es una etiqueta que poco a poco se intenta que sustituya a <div> , su uso es para dividir la pagina en secciones y agrupar contenido relacionado
<article> Es una etqueta que se utiliza para agrupar contenido autonomo, que si se separa de otro contenido sigue teniendo coherencia. Normalmente se compone de un encabezado y contenido
<footer> Es una etiqueta que normalmente se encuentra al final de una pagina y funciona como cierrre de un sitio web. A menudo contiene datos de contacto, derechoos de autor u otra informacion complementaria.

Etiqueta principales HTML

Etiqueta <a>

Es unb elemento que nos va a permitir navegar a:

  • Páginas externas
  • Páginas internas
  • Secciones dentro de la misma página (enlaces ancla)

Algunos valores especifiocs para el atributo son:

href="https://wa.me/5583386104"
Abre un chat del número especificado en WhastApp
href="mailto.example@.com
Abre una aplicación de correo, con el destinatario especificado

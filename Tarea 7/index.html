Formas de Agregar Estilos a nuestra Pagina Web

Estilos en Linea

Estos estilos se agregan directamente en el elemento html, mediante el atributo style

Este parrafo tiene estilos en linea.

Estilos Internos

Estos estilos se agregan en el mismo archivo html, dentro de la etiqueta <head> y se utiliza la etiqueta <style>

Estilos Externos

Estos estilos son los que se recomiendan, ya que ofrecen modularidad, facil manetnimiento, orden, eficiencia, entre otras ventajas, para agregar una hoja de estilos externa, dentrp de la etiqueta <head> se va a utilizar la etiqueta <link>
